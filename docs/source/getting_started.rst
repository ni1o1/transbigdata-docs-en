安装
=============================

TransBigData依赖geopandas：https://geopandas.org/index.html
如果你已经安装了geopandas，则直接在命令提示符中运行下面代码即可安装::

  pip install -U transbigdata

在Python中运行下面代码::

  import transbigdata as tbd

依赖包
=============================
TransBigData依赖如下包

* pandas
* geopandas
* scipy
* matplotlib

可选依赖：

* networkx
* keplergl
